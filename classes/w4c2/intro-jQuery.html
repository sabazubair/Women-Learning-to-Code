Intro to jQuery

  • Think of jQuery as a collection of helpful pre-written JavaScript code that you can use to quickly solve some common front-end problems. Behind the scenes, it's all javascript.
  • Why learn jQuery? Why not stick to vanilla JS? jQuery provides shortcuts, irons out a lot of cross-browser inconsistencies, and lots of companies are still using jQuery!
  • Aside: We wanted you to get exposure using a library. I showed you bootstrap when we were completing CSS but there wasn't enough time to delve into CSS libraries. Since our focus is largely JS, we wanted to teach you jQuery.
  • As a dev, your job will be to write code + maintain legacy code.
  • jQuery is one biiiig object. It's got a lot of helpful methods that we can use. What's a method? Let's review below.
  • We'll use jQuery to make requests, submit forms + display output, plus events.
  • We're combining all that we've learned! HTML + CSS + Javascript! Woohoo!
