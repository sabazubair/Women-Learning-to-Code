Arrays

  • often described as a list; official definition: list-like object
  • square brackets; we refer to items in arrays as "elements"
  • i.e. [elem1, elem2, elem3]
  • arrays are indexed; start at 0
  • contain any type of element: arrays can have arrays, numbers, strings, booleans, objects. Example: let arr = [0, false, "hello", [], ...]
  • Array length is variable. In other words: arrays can be empty, or be 4.29 billion elements max. (I checked!)
  • TL;DR; array is a powerful data structure (or: a way to store data)!

Loops

  • loops are commonly used with arrays; hence, why we taught them together. Last class, you learned for loops & while loops.
  • Loops check a condition. If it returns true, a code block will run. Then the condition will be checked again and if it still returns true, the code block will run again.
  • It repeats until the condition returns false.
  • There are 5 different loops in Javascript - for, while, do...while, for...in, for...of. If you understand the top two, you'll have no problem understanding the bottom 3. TBH: I've only ever used the for loop out of this list.
  • The most common loop in Javascript is the for loop.
  • for loop, from the onset, is made up of 3 statements; initialization, condition, update.
  • "for" keyword, (let i=0 ...) is the initialization; creates variable i; sometimes called index.
  • for (let i = 0; i < 10 ...) - is the condition; the value of i was initially set to 0, so in this case, the loop will run 10 times before stopping
  • for (let i=0; i < 10; i++) - update statement; how would you like the loop to proceed. There is also the decrement operator.
  • for (i=0; i > 10; i++) {code block} - what you'd like to occur every time the loop runs
  • The while loop loops through a block of code as long as a specified condition is true.
  • IRL, you'll be most encouraged to use: map, reduce, filter to loop, or "iterate" + perform a transformation on arrays. They are similar to for and while loops.
